Spider Webs

Call blocking service for android phones

    Main activity: Black list process

  • Phone call comes in

  • Broadcast reciever alerts phone state listener

  • Phone state listener checks if phone is ringing

  • Ringer muted

  • Phone number parsed and hashed

  • Hash table is searched for match

  • Call denied if match is found

  • Ringer unmuted

  • Normal operation resumes

    Design justificaitons

  • Broadcast reciever allows for the app to run in background.

  • 251 entry hash table for look ups; quick searches

  • 251 is the largest prime before 255
